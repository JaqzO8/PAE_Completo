Auth Service (puerto 3001):

POST    /login              # Login email/password
POST    /google             # Login con Google (RQ58)
POST    /refresh            # Refresh token
POST    /logout             # Cerrar sesión (RQ107)
POST    /logout-all         # Cerrar todas las sesiones (RQ107)
POST    /forgot-password    # Recuperación (RQ96)
POST    /reset-password     # Reset password
POST    /verify-2fa         # Verificar 2FA (RQ95)
POST    /enable-2fa         # Activar 2FA
GET     /sessions           # Obtener sesiones activas (RQ101, RQ104)
DELETE  /sessions/:id       # Cerrar sesión específica
POST    /verify-token       # Para que user-service verifique tokens
